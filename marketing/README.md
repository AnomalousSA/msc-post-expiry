## Article Template

**File**: `articles/ARTICLE_TEMPLATE.md`

**Purpose**: Template for creating new marketing articles

**Usage**:
1. Copy this template
2. Rename to your article name
3. Fill in the frontmatter (title, description, keywords, etc.)
4. Write your article content
5. Follow the structure and formatting

**Frontmatter Fields**:
- `title`: Article title (50-60 characters)
- `description`: Meta description (150-160 characters)
- `keywords`: SEO keywords (comma-separated)
- `author`: Author name
- `date`: Publication date (YYYY-MM-DD)
- `category`: Article category
- `plugin`: Plugin name

## Content Guidelines

### Writing Style

- **Clear and concise**: Use simple language
- **Practical**: Focus on real-world use cases
- **Helpful**: Provide actionable advice
- **Professional**: Maintain professional tone
- **Scannable**: Use headings and bullet points

### Structure

1. **Introduction**
   - Hook the reader
   - Explain the problem
   - Introduce the solution

2. **What is [Plugin]?**
   - Brief overview
   - Key features
   - Benefits

3. **Installation**
   - Step-by-step instructions
   - Multiple methods if applicable
   - Verification steps

4. **Configuration**
   - Basic setup
   - Common options
   - Explanations

5. **Use Cases**
   - Real-world examples
   - Recommended setups
   - When to use each option

6. **Advanced Features**
   - Pro features
   - Customization options
   - Developer information

7. **Troubleshooting**
   - Common problems
   - Solutions
   - When to get help

8. **Best Practices**
   - Tips and tricks
   - Optimization
   - Performance

9. **Getting Help**
   - Support options
   - Documentation links
   - Contact information

10. **Next Steps**
    - What to do next
    - Related articles
    - Call-to-action

### SEO Optimization

- **Title**: Include main keyword (50-60 characters)
- **Description**: Include main keyword (150-160 characters)
- **Headings**: Use H2 and H3 for structure
- **Keywords**: Include naturally throughout
- **Links**: Link to related content
- **Images**: Use descriptive alt text

### Formatting

- **Headings**: Use H1 for title, H2 for sections, H3 for subsections
- **Lists**: Use bullet points for lists
- **Code**: Use code blocks for examples
- **Emphasis**: Use **bold** for important terms
- **Links**: Use [text](url) format
